ОПИСАНИЕ ПРОЕКТА
Клиент отпраляет на сервер по TCP матрицу 6х6 с числами типа double с интервалом 2 секунды.
Сервер вычисляет определитель принятой матрицы, среднее арифметическое среди 
последних полученных пяти матриц и определительй матрицы, полученной пять матриц  назад.
И выводит эти данные в консоль.

СБОРКА И ЗАПУСК ПРОЕКТА

В одном терминале:
0. cd ClientServer
1. make
2. ./server

В новом терминале:
3. cd ClentServer
4. ./client
